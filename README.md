# Valdoz Optic E-commerce Website

## 👓 About
Valdoz Optic is a premium eyewear e-commerce platform based in Douala, Cameroon. We provide quality glasses, sunglasses, and contact lenses with professional eye care services.

## 🚀 Features
- Responsive design for all devices
- Product catalog with categories
- Shopping cart functionality
- Blog system for eye care tips
- Contact form with email notifications
- Admin dashboard for management
- Secure user authentication
- Product search and filtering

## 🛠️ Technologies Used
- PHP 7.4+ / 8.x
- MySQL / MariaDB
- Bootstrap 5
- Font Awesome 6
- JavaScript (Vanilla)
- AOS Animation Library

## 📋 Requirements
- Web server (Apache/Nginx)
- PHP 7.4 or higher
- MySQL 5.7 or higher
- mod_rewrite enabled (for clean URLs)
